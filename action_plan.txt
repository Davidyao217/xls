Here's the prompt to pass:

---

Build a single `index.html` file — a browser-based spreadsheet for quick arithmetic scratchwork. Target total payload under 15KB. No frameworks, no external JS, no build step.

**Grid:** ~20 rows × 10 columns (A–J). Enough for scratchwork, not a full spreadsheet.

**Formula engine:**
- Cell references: `A1`, `B3`
- Ranges: `A1:B5`
- Arithmetic: `+`, `-`, `*`, `/`, parentheses
- Functions: `SUM`, `AVERAGE`, `MIN`, `MAX`, `COUNT`
- Circular reference detection (show error in cell)

**UI:**
- Formula bar at top showing raw formula of selected cell
- Click or arrow-key navigation between cells
- Enter commits and moves down, Tab commits and moves right, Escape cancels edit
- Double-click or typing enters edit mode
- Cells display computed values; formula bar shows the formula

**CSV export:** Button that exports computed values, auto-bounded to the used range (no trailing empty rows/cols). Downloads as `.csv`.

**Constraints:**
- Single HTML file, all CSS/JS inline
- No external dependencies except a single non-render-blocking Google Font (Inter or similar, via `<link rel="preconnect">` + `display=swap`)
- Clean, minimal visual design — light gray gridlines, monospace cells, no chrome
- No persistence, no undo, no mobile optimization

**Repo structure:**
```
index.html
_headers
```

`_headers` contents:
```
/
  Cache-Control: public, max-age=31536000, immutable
```

This deploys to Cloudflare Pages with no build step, build output directory `/`.
