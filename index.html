Formulas

Start a cell with = to write a formula. Use cell refs like A1 or ranges like A1:C3.

=SUM(A1:A10)Sum of a range
=AVERAGE(A1:A10)Average of a range
=MIN(A1:A10)Minimum value
=MAX(A1:A10)Maximum value
=COUNT(A1:A10)Count of numeric cells
=ABS(A1)Absolute value
=TODAY()Today's date
